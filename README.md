# learn-polymer-starter-kit
A starter kit app in Polymer 3. This repository was made in conjuction with this [blog post](http://kasey-hinton.surge.sh/polymer-routing).

### Quickstart
```bash
  npm install -g polymer-cli
  npm install
  polymer serve
```
